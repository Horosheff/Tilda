Как использовать

  1. Получите API ключ Gemini и сохраните его выше
  2. Откройте редактор страницы в Tilda
  3. Нажмите кнопку ✨ в правом нижнем углу
  4. Опишите нужный блок или страницу
  5. Скопируйте HTML и вставьте в блок T123
