Desktop control center

Tilda Space AI

Взрослый control room для генерации страниц: модель, шаблон, SVG, пресеты, parallel block agents и ручная вставка блоков в Tilda.

1 orchestrator создаёт общий дизайн и блок-план
N agents отдельно генерируют каждый блок
Ordered queue вставляет HTML в правильном порядке
