# Готовые сценарии · Дизайн-тренды 2026

Сценарии построены на анализе рынка: [Moburst](https://www.moburst.com/blog/landing-page-design-trends-2026/), [B12](https://www.b12.io/resource-center/website-design/nine-landing-page-trends-to-watch-for/), [Graphic Design Junction](https://graphicdesignjunction.com/2026/01/landing-page-design-trends-2026/), [Neo-Brutalism](https://www.graphiceagle.com/neubrutalism-bold-and-raw-interfaces-redefining-ui-design-in-2025/), [Bento Grid](https://landdding.com/blog/blog-bento-grid-design-guide).

## Ключевые тренды 2026

| Тренд | Суть | Применение |
|-------|-----|------------|
| **Clarity over decoration** | Скорость и ясность важнее визуальных излишеств | Короткие блоки, один CTA, минимум текста |
| **Bento Grid** | Асимметричные модули как у Apple (67% SaaS на ProductHunt) | Feature showcase, hero, dashboard |
| **Neo-Brutalism** | Жирные контуры, контраст, raw-эстетика, Gen Z | Брендинг, креатив, выделение |
| **Glassmorphism** | Полупрозрачность, blur, слои | Премиум, утончённость |
| **Earth Tones** | Тёплые землистые цвета, текстура, доверие | Человечность, warmth |
| **Dark Tech** | Linear, Vercel — тёмный, минимализм | IT, SaaS, dev tools |
| **Micro-interactions** | Hover, scroll, motion без перегрузки | Любой лендинг |
| **Mobile-first** | ~83% трафика с телефона | Все сценарии |

## Сценарии в приложении

### 🔮 KV-AI Premium
Ваш эталон: фиолетовый градиент, желтый акцент. AI-сервис, премиум.

### 📦 Bento Grid SaaS
Сетка модулей разного размера. Hero + 5–8 секций (2x2, 1x2, 2x1). Размер = иерархия. Тёмный фон, синий акцент.

### ⚡ Neo-Brutalism
Жирные контуры 3–4px, жёлтый + чёрный, flat blocks, асимметрия. Raw, Gen Z.

### 🪟 Glassmorphism
Backdrop-blur, полупрозрачные карточки. Тёмно-синий, индиго акцент. Слои, глубина.

### 🌿 Earth Tones
#8B7355, #D4A574, кремовый. Текстура, тепло, humanity over minimalism.

### 🚀 Dark Tech
#09090b, зелёный/фиолетовый акцент. Linear/Vercel style. IT, dev tools.

### 🎨 Креатив-агентство
Портфолио + услуги. Крупная типография, галерея, смелые заголовки.

### ☕ Кофейня
Уют, тёплые коричневые, фото, меню. Рукописные акценты.

### 💪 Фитнес
Энергия, оранжевый/красный акцент. Тренеры, направления, тарифы.

### 📊 Yandex Metrika Style
Корпоративный стиль портала [yandex.ru/adv/metrika](https://yandex.ru/adv/metrika). Hero + stat-badge, feature-duo, grid возможностей, stats 3 числа, news, promo-block тёмный, «Читайте также», FAQ accordion, CTA 2 кнопки, footer. Цвета: #fff, #2D7FF9, #000, #666. Карточки: radius 12px, shadow. Чистый, без градиентов.

## 📊 Yandex Metrika Style

**Референс:** [yandex.ru/adv/metrika](https://yandex.ru/adv/metrika) — портал продукта Яндекс Метрика.

### Структура блоков (по порядку)

| # | Блок | Описание |
|---|------|----------|
| 1 | **Hero** | H1 (40–48px) + subline + primary CTA «Создать»/«Начать» + stat-badge («95% используют») |
| 2 | **Feature-duo** | 2 большие карточки 50/50: иконка 32px + H3 + 2–3 строки текста |
| 3 | **Grid «Возможности»** | 5–6 карточек-ссылок в сетке: иконка + заголовок + 1 строка |
| 4 | **Stats** | 3 крупных числа в ряд (9 млн, 1.6 млн, 120) + подписи |
| 5 | **News** | 3 карточки: дата + заголовок + excerpt + «Подробнее»/«Больше» |
| 6 | **Promo-block** | Тёмный блок (bg #1a1a1a): заголовок + описание + CTA «Подробнее» |
| 7 | **«Читайте также»** | 6–8 карточек статей: thumbnail, заголовок, дата, excerpt, тег |
| 8 | **FAQ «С чего начать»** | Accordion 4–5 вопросов с развёрнутым ответом |
| 9 | **CTA** | 2 кнопки рядом: primary «Создать» + secondary «Заказать настройку» |
| 10 | **Footer** | 3 быстрые ссылки (иконка+текст) + мега-колонки с навигацией |

### Design System

| Параметр | Значение |
|----------|----------|
| **bg** | #ffffff |
| **bgAlt** | #f5f5f6 |
| **primary** | #2D7FF9 (Yandex blue) |
| **text** | #000000 |
| **textMuted** | #666666 |
| **font** | Yandex Sans, -apple-system, system-ui |

### Карточки

- `border-radius: 12px`
- `box-shadow: 0 2px 8px rgba(0,0,0,0.08)`
- `padding: 24px`

### Кнопки

- **Primary:** `background: #2D7FF9`, белый текст
- **Secondary:** outline или текстовая ссылка

### Стиль

Чистый корпоративный. Без тяжёлых градиентов, без декора. Чёткая иерархия, читаемая типография, air между секциями.
