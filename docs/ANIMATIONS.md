# Дизайнерские анимации Tilda Space AI

Расширение добавляет галочки для **дизайнерских** анимаций — эффекты уровня премиум-лендингов. Чистый CSS и JavaScript, без jQuery/GSAP.

## Типы анимаций (в панели)

### Появление при скролле
| Опция | Описание | Эффект |
|-------|----------|--------|
| **Каскад (stagger reveal)** | Элементы появляются каскадом с easing cubic-bezier | translateY(40px) + opacity, задержка по индексу |
| **Fade In Up** | Плавное появление снизу вверх | translateY(28px) + opacity |
| **Zoom In** | Масштабирование при появлении | scale(0.92→1) + opacity |
| **Text clip reveal** | Заголовки h1-h4 раскрываются слева направо | clip-path: inset(0 100% → 0 0) |

### Hover-эффекты
| Опция | Описание | Эффект |
|-------|----------|--------|
| **Card lift + тень** | Карточки приподнимаются с объёмной тенью при hover | translateY(-8px) + box-shadow |
| **Glow при hover** | Свечение акцентного цвета (#694be8) при наведении | box-shadow с размытием |
| **Tilt** | Лёгкий 3D наклон при наведении | perspective + rotateX/Y + scale |

### Фоновые эффекты
| Опция | Описание | Эффект |
|-------|----------|--------|
| **Параллакс** | Лёгкое смещение при скролле (разная скорость) | translate3d + --py CSS variable |
| **Float** | Тонкое вертикальное покачивание | animation translateY(-4px) цикл 4s |

## Технологии

- **Easing:** cubic-bezier(0.22, 1, 0.36, 1) — премиальный «overshoot»
- **Intersection Observer** — scroll reveal без тяжёлых библиотек
- **CSS custom properties** — параллакс совместим с card lift
- **prefers-reduced-motion** — анимации отключаются для чувствительных пользователей

## Совместимость

Эффекты можно комбинировать: например, каскад + card lift + glow для карточек премиум-лендинга.
